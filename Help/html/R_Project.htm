DTIControls Documentation
The DTIControls Suite allows any developer to add content management to any web application with little or no impact to existing code.

Namespaces

Namespace Description
BaseClasses
Base classes for most needed objects adding a data access layer and virtual path management to existing objects.
ckEditor
A WYSIWYG html editor
DTIAdminPanel
Adds dynamic layout and dynamic pages to any site.
DTIContentManagement
Contains the EditPanel, a drop in replacement for any div to add content management.
DTIGallery
A Dynamic Gallery control
DTIGoogleCalendar
The Google calander as a dragable dynamic control.
DTIImageManager
Adds dynamic images to content management. Allows user to upload any size image and resize it withing the editor.
DTIMapControl
The Google map as a dragable dynamic control.
DTIMediaManager
Contains generic searchers and uploaders for any media type.
DTIMiniControls
Any basic control that is reused across many componets is here.
DTIServerControls
The Base classes of any data-centric control starts here
DTISortable
Adds dynamic layout to content management allowing you to Drag controls from one sortable are to another.
DTITagManager
Adds tags to ages and media for searching and search optimization.
DTIUploader
Upload handler namespace for images, videos and other files.
DTIVideoManager
Video uploaders converters and players. Adds video upload capabilities to the content management system.
jQueryLibrary
Add jquery to any page with a single line of code. Also adds themeing if controlls use jquery themes.
Rotator
A rotating edit area as a dragable dynamic control.
SQLiteHelper
adds SQLite as an optional data connector for the data access layer..
VideoPanel
A video playback area as a dragable dynamic control.
